Responzivna slika - Primjer

1. Osnovni responzivni prikaz (CSS)

img {
  max-width: 100%;
  height: auto;
}

2. Responzivna slika s srcset atributom

<img src="primjer-400.jpg"
     srcset="primjer-800.jpg 800w, primjer-1200.jpg 1200w"
     sizes="(max-width: 600px) 480px, 800px"
     alt="Primjer slike s srcset atributom"
     loading="lazy">

3. Element <picture> s modernim formatima

<picture>
  <source srcset="primjer.webp" type="image/webp">
  <source srcset="primjer.avif" type="image/avif">
  <source srcset="primjer.jpg" type="image/jpeg">
  <img src="primjer.jpg" alt="Primjer elementa picture" loading="lazy">
</picture>

4. Retina prikaz (visoka rezolucija)

<img src="primjer.jpg"
     srcset="primjer.jpg 1x, primjer-2x.jpg 2x"
     alt="Primjer za Retina ekrane"
     loading="lazy">
