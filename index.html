Responsive Image Architecture

Demonstracija modernih metoda optimizacije slika za web razvoj.

CSS

01. Fluid Layout

Osnovna responzivnost osigurava da slika nikada ne "pukne" izvan okvira svog roditelja.

img {
  max-width: 100%;
  height: auto;
  display: block;
}
HTML5

02. Smart Density (srcset)

Preglednik sam bira veličinu slike ovisno o rezoluciji ekrana (štedi podatke na mobitelu).

<img src="small.jpg" 
     srcset="med.jpg 800w, large.jpg 1200w" 
     sizes="(max-width: 600px) 100vw, 800px">
Optimization

03. Next-Gen Formats

Korištenje <picture> taga za serviranje WebP ili AVIF formata uz fallback na JPG.

<picture>
  <source srcset="img.webp" type="image/webp">
  <img src="img.jpg" alt="Fallback">
</picture>
